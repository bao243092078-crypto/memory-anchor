# 北极星 | Memory Anchor

## 我要做什么（一句话）
为 AI 编程助手（Claude Code / Cursor / Codex）提供**跨会话持久化记忆**，解决上下文压缩导致的"失忆"问题。

## 核心隐喻
把 AI 当作阿尔茨海默症患者——能力很强，但容易因上下文压缩而"失忆"。Memory Anchor = AI 的外挂海马体。

## 解决的 AI 问题
1. **上下文压缩失忆**：聊到一半，压缩后忘了之前讨论的细节
2. **任务遗忘**：做着做着忘了主要目标，丢三落四
3. **幻觉跑偏**：自己编造任务，跑去做别的事情

## 绝对不做
- 不为 AI 增加认知负担（工具要简单）
- 不依赖 AI 自觉（要有强制机制）
- 不牺牲简单性换功能
- 不让开发者手动管理底层存储

## 怎么算完成
- AI 上下文压缩后能恢复关键决策 = 成功
- 每次会话开始自动加载项目上下文 = 成功
- 开发者说"终于不用重复解释了" = 成功

## 核心工具
| 工具 | 作用 |
|------|------|
| `search_memory` | 检索历史决策、Bug 修复记录 |
| `add_memory` | 存储重要发现、任务完成记录 |
| `get_constitution` | 加载项目核心身份（始终预加载） |
